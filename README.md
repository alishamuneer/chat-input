# ChatInput

## Environments

```
Development

==> Branch Name: main

```

## Install

```sh

git clone https://github.com/alishamuneer/chat-input.git
cd chat-input
npm install
```

## Start development

```sh
npm run watch
```

## Build for production

```sh
npm run build
```
